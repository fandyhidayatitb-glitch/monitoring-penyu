🐢 Sarang Siap Menetas / Pemeliharaan
🌊 Log Pelepasan Tukik
📋 Protokol Pelepasan (SOP IOSEA/CMS)
✅ Sebelum Pelepasan
  • Pilih waktu pelepasan sore/malam hari untuk menghindari panas matahari
  • Pastikan pantai bebas dari predator dan gangguan manusia
  • Hindari penggunaan senter putih — gunakan cahaya merah jika diperlukan
  • Jangan berjalan di antara tukik dan laut saat mereka bergerak
  • Pastikan tukik bergerak sendiri ke arah laut (imprinting alami)
📏 Kondisi Optimal
  • Suhu air pemeliharaan: 26–29°C
  • Waktu terbaik: malam hari saat pasang naik
  • Jarak pelepasan: minimal 5 meter dari bibir pantai
  • Maksimal 3 orang selama proses pelepasan
🚫 Yang Harus Dihindari
  • Jangan gunakan lampu/cahaya putih atau api di dekat tukik
  • Jangan bawa tukik ke air secara paksa — biarkan berjalan sendiri
  • Jangan lepaskan saat ada turis/keramaian yang tidak terkontrol
  • Jangan gunakan tas plastik atau wadah yang bisa mencekik
  • Hindari pelepasan saat cuaca ekstrem (hujan lebat, ombak besar)
📝 Dokumentasi Wajib
  • Catat jumlah tukik yang dilepas via KoboCollect
  • Foto/video pelepasan wajib (upload ke form)
  • Catat waktu, cuaca, dan metode pelepasan
  • Input GPS lokasi pelepasan
SOP & Panduan Monitoring (IOSEA/CMS)
🐾 Pengenalan Jejak & Spesies
Penyu Hijau (Chelonia mydas)
  • Jejak: dalam, simetris, paralel
  • Karapas: hijau kecoklatan, halus, oval
  • Ukuran: 90–120 cm; berat: 68–190 kg
  • Waktu inkubasi: 45–75 hari (rata-rata 60 hari)
Penyu Belimbing (Dermochelys coriacea)
  • Jejak: sangat lebar, bekas perut khas
  • Karapas: hitam seperti kulit, tanpa sisik keras
  • Ukuran hingga 180 cm; terbesar di dunia
  • Waktu inkubasi: 60–70 hari
Penyu Sisik (Eretmochelys imbricata)
  • Jejak: asimetris/zigzag, lebih sempit
  • Karapas: unik seperti sisik tumpang tindih
  • Ukuran: 60–90 cm
Penyu Lekang (Lepidochelys olivacea)
  • Jejak: asimetris/zigzag, mirip sisik
  • Karapas: olive-abu, hampir bundar
  • Ukuran: 60–70 cm

🔗 Integrasi KoboCollect — Panduan Singkat
Cara Menghubungkan KoboToolbox API
  • Masuk ke koba.humanitarianresponse.info atau kobo.unhcr.org
  • Buka profil → API token (terlihat di bawah username)
  • Copy token dan paste di tab Pengaturan dashboard ini
  • Masukkan Asset UID form (ada di URL saat membuka form)
  • Klik "Test Koneksi" lalu "Simpan & Aktifkan Auto-Sync"
Format Data yang Didukung
  • KoboToolbox REST API v2 (JSON)
  • ODK Central REST API
  • CSV/Excel import manual (via tombol Import)
  • Google Sheets (via Apps Script webhook)
Field Mapping XLSForm → Dashboard
  • kode_sarang → ID Sarang
  • jenis_penyu → Spesies
  • jumlah_telur_awal → Total Telur
  • suhu_sarang → Suhu Inkubasi
  • status_penetasan → Status Sarang
  • jumlah_tukik_menetas → Tukik Menetas
  • jumlah_tukik_dilepas → Tukik Dilepas
  • gps_lokasi → Koordinat Peta
